1 Test Runner GUI
Графическое приложение для запуска тестов проекта с использованием pytest
2 Возможности
- Обнаружение всех тестов в каталоге `tests/`.
- Выбор тестов для запуска.
- Выполнение тестов в отдельном потоке (GUI не зависает).
- Цветовое выделение результатов (PASSED/FAILED).
- Прогресс-бар.
- Отображение полного вывода pytest, включая traceback ошибок.
3 Требования
- Python 3.7+
- Установленные зависимости: `pytest`, `pytest-cov`, `pyinstaller` (только для сборки).
4 Установка
1. Клонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv venv`
3. Активируйте его.
4. Установите зависимости: `pip install -r requirements.txt`

5 Использование
Запустите приложение:
```bash
python gui/test_runner.py
